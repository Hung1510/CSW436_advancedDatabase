OLAP Operations (Aggregation in MongoDB): Use the students collection for the following aggregation queries. 
1.	Use - Group By: write an aggregation query to calculate the average marks of students in each class. 
db.students.aggregate([
  {
    $group: {
      _id: "$class",
      averageMarks: { $avg: "$marks" }
    }
  }
])

2.	Sum of Marks: write an aggregation query to calculate the total marks of students in each class. 
db.students.aggregate([
  {
    $group: {
      _id: "$class",
      totalMarks: { $sum: "$marks" }
    }
  }
])

3.	Write an aggregation query to count the number of students in each city. 
db.students.aggregate([
  {
    $group: {
      _id: "$city",
      studentCount: { $count: {} }
    }
  }
])

4.	Filter and Group - write an aggregation query to: 
•	Find the average marks of students from the city "Binh Doung". 
db.students.aggregate([
  {
    $match: { city: "Binh Duong" }
  },
  {
    $group: {
      _id: "$city",
      averageMarks: { $avg: "$marks" }
    }
  }
])

5.	Sort and top results - write an aggregation query to: 
•	Display classes sorted by average marks in descending order. 
db.students.aggregate([
  {
    $group: {
      _id: "$class",
      averageMarks: { $avg: "$marks" }
    }
  },
  {
    $sort: { averageMarks: -1 }
  }
])
